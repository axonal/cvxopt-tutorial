# CVXOPT Tutorial

## Installation
```pip install -r requirements.txt```

Optionally install [glpk](https://www.gnu.org/software/glpk/):
```brew install glpk```

or take a look at the [installation instructions](cvxopt.org/install/index.html).


## Run code
```python clrs.py```

```python chicken.py```
